# 涉诈APP智能识别分析系统

本文件是涉诈APP智能识别分析系统的使用介绍文件。系统采用多模态模型、深度学习模型、Vue.js、SpringBoot、数据库、云服务器等技术，是一款旨在判别APP是否安全、并进一步判断危险类型和进行深度解析的项目。系统功能全面，操作简单，适用于各种用户群体。

## 前端说明 ( 若无需本地下载源码运行，则可跳过 )

前端使用vue3 (Composition API) + vite

### 下载项目配置

```sh
npm install
```

### 开启项目

```sh
npm run dev
```

### 打包项目

```sh
npm run build
```
